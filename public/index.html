5. Sort Events

Endpoint: GET /api/events/sort-by?criteria=<criteria>&order=<order>

Sort events based on the provided criteria and order.

Example Request:
fetch('/api/events/sort-by?criteria=date&order=asc').then(response => response.json()).then(data => console.log(data));

6. Get Event Details

Endpoint: GET /api/events/:id

You can retrieve event details using either the _id or the event_id of the event. Provide the corresponding value in the URL parameter.

Example Request:

        
Using id
 
        fetch('/api/events/64b2e8a9d3c7e53c9d0e4a9b')
        .then(response => response.json())
        .then(data => console.log(data));
        
        
Using event_id
 
        fetch('/api/events/1003')
        .then(response => response.json())
        .then(data => console.log(data));
Response Example:
