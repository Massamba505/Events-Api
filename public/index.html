Events and Activities API Documentation 3A

A guide to using the Event API endpoints for managing and retrieving event data.

Available Endpoints

1. Get All Events

Endpoint: GET /api/events

Retrieve all events from the database.

Example Request:
fetch('/api/events').then(response => response.json()).then(data => console.log(data));
Response Example:

3. Get In-Progress Events

Endpoint: GET /api/events/inprogress-events

Retrieve events that are currently in progress.

Example Request:
fetch('/api/events/inprogress-events').then(response => response.json()).then(data => console.log(data));

5. Sort Events

Endpoint: GET /api/events/sort-by?criteria=<criteria>&order=<order>

Sort events based on the provided criteria and order.

Example Request:
fetch('/api/events/sort-by?criteria=date&order=asc').then(response => response.json()).then(data => console.log(data));

6. Get Event Details

Endpoint: GET /api/events/:id

Retrieve details for a specific event based on its ID.

Example Request:
fetch('/api/events/12345').then(response => response.json()).then(data => console.log(data));
Response Example:
