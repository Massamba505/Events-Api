Event API Endpoints

This endpoint retrieves a list of all events available in the system.

{
  "events": [
    {
      "event_id": "12345",
      "eventAuthor": "John Doe",
      "email": "johndoe@example.com",
      "title": "Tech Conference",
      "description": "A technology event.",
      "location": "City Hall",
      "date": "2024-12-10",
      "start_time": "09:00 AM",
      "end_time": "05:00 PM",
      "is_paid": true,
      "ticket_price": 50,
      "max_attendees": 100,
      "current_attendees": 80,
      "category": "Technology",
      "images": ["img url", "img url"]
    }
  ]
}
